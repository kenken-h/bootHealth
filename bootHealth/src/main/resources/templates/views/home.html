これは、Spring Boot + Thymeleaf + Bootstrap
で開発したデモアプリケーションです。


開発環境：
  • OSX 10.10.5, JDK1.8, Eclipse Mars, MySQL5.6.17, Maven 3.1.1
  • Spring Boot1.2.3, Thymeleaf2.1.4, Bootstrap3.3.2, jQuery1.11.1, dataTables1.10.4, EclipseLink2.5.0
実行環境：
  • サーバー　　(Macbook pro)：OSX 10.10.5, JVM1.8, MySQL5.6.17
  • クライアント(Macbook pro)：OSX 10.10.5, Safari8.0.8
  • クライアント( iPad2 )：iOS9, Safari9.0
Spring Boot Actuator を使ってサーバーの状態を表示：
  • コンテナに登録されているビーンの一覧
  • プロパティ一覧（ConfigurableEnvironment）
  • プロパティ一覧（@ConfigurationProperties）
  • マッピング情報
  • アプリケーションのメトリックス情報
  • トレース情報
  • 自動設定の候補
  • スレッド・ダンプ
  • サーバーのヘルスチェック
